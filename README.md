# 花札役ガイド — Hanafuda Yaku Guide

花札こいこいの役をサッと確認できる iOS アプリ。  
Nintendo Museumなどで花札を遊ぶ時に、隣に置いて使えるシンプルなガイドアプリです。

---

## 概要

- **とった札をタップ** → 自動で役を判定
- 12ヶ月 × 4枚 = **全48枚の花札**を月ごとに整理
- **グレーアウト**で選択済み札を視覚的に確認
- こいこいの**全12役**に対応
- レトロでモダンなデザイン

## セットアップ

### 必要環境

- macOS 13.0 以上
- Xcode 15.0 以上
- iOS 17.0 以上（実機またはシミュレーター）

### ビルド方法

```bash
# リポジトリをクローン
git clone https://github.com/manabe47/hanafuda-yaku-guide.git
cd hanafuda-yaku-guide

# Xcodeで開く
open HanafudaYakuGuide.xcodeproj
```

Xcode でプロジェクトを開き、シミュレーターまたは実機を選択して ▶️ ボタンでビルド・実行してください。

## 使い方

1. 画面上部の **月タブ**（松〜桐）をタップして月を選択
2. 下のカードグリッドで **取った札をタップ** → グレーアウトで選択
3. 画面下部の **成立役パネル**に、揃った役と点数が表示される
4. 右上の **↺ ボタン**でリセット
5. 右上の 📋 ボタンで **役一覧**を確認
